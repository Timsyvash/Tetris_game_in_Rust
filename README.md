# Tetris_game_in_Rust  
Я створив гру Тетріс не Rust.  
## Що таке Тетріс?  
Тетріс — відеогра-головоломка, створена 1984 року радянським програмістом Олексієм Пажитновим з колегами. Першу версію було представлено 6 червня 1984 року, коли Олексій працював в ОЦ Академії наук СРСР у Москві. ␠␠
Назву гри автор створив поєднавши грецький префікс «тетра-» зі словом «теніс» — улюбленою грою Пажитнова[2]. Гра в різних інтерпретаціях існує майже для кожної ігрової консолі та операційної системи, а також для інших пристроїв: мобільних телефонів, медіаплеєрів,
кишенькових комп'ютерів тощо.  
## Що таке Rust?  
Rust (укр. Раст, досл. «Іржа») — сучасна багатопарадигмова мова програмування загального призначення. Мова має сильну (строгу) типізацію і зосереджена на безпечній роботі з пам'яттю та забезпеченні високої рівночасності виконання задач (можливість породжувати тисячі й навіть мільйони підпроцесів). Знаючи Rust, можна працювати як розробник програмного забезпечення у сферах системного програмування (ОС, драйвери), веб-розробки (бек-енд, WebAssembly), блокчейн, ігрового девелопменту (геймдев), вбудованих систем (IoT), машинного навчання та розподілених систем. Це дозволяє створювати високопродуктивні, безпечні та надійні додатки для широкого спектру завдань. Rust використовується для системного програмування, веб-розробки (бек- та фронтенд), блокчейну, розробки ігор, вбудованих систем (IoT) та інструментів із високими вимогами до продуктивності та безпеки, замінюючи C/C++ у проєктах, де важлива надійна робота з пам'яттю та паралелізм, як-от операційні системи (Redox), браузери (Firefox, Brave), криптовалюти (Bitcoin, Ethereum) та високо навантажені сервіси (Discord, Dropbox), а також завдяки WebAssembly.
